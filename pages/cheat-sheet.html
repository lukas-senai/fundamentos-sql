Cheat Sheet - Referência Rápida

Uma lista rápida dos comandos SQL mais utilizados para consulta durante seus estudos e desenvolvimento.

Definição de Dados (DDL)

Comando Descrição Exemplo
CREATE TABLE Cria uma nova tabela 
CREATE TABLE Pessoas (
id INT PRIMARY KEY,
nome VARCHAR(100)
);
ALTER TABLE Modifica a estrutura de uma tabela 
ALTER TABLE Pessoas
ADD idade INT;
DROP TABLE Exclui uma tabela 
DROP TABLE Pessoas;

Manipulação de Dados (DML)

Comando Descrição Exemplo
SELECT Recupera dados de uma tabela 
SELECT nome, idade
FROM Pessoas;
INSERT Insere novos dados em uma tabela 
INSERT INTO Pessoas (nome, idade)
VALUES ('João', 25);
UPDATE Atualiza dados existentes em uma tabela 
UPDATE Pessoas
SET idade = 26
WHERE nome = 'João';
DELETE Exclui dados de uma tabela 
DELETE FROM Pessoas
WHERE idade > 30;

Cláusulas e Funções

Comando Descrição Exemplo
WHERE Filtra registros com base em condições 
SELECT * FROM Pessoas
WHERE idade > 18;
ORDER BY Ordena os resultados 
SELECT * FROM Pessoas
ORDER BY nome ASC;
GROUP BY Agrupa registros com valores iguais 
SELECT idade, COUNT(*)
FROM Pessoas
GROUP BY idade;
JOIN Combina registros de duas ou mais tabelas 
SELECT p.nome, c.nome
FROM Pessoas p
INNER JOIN Cidades c
ON p.cidade_id = c.id;

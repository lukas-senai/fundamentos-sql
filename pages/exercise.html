Exercícios e Quiz

Teste seus conhecimentos de SQL com estas questões. Ao final, você receberá um resumo do seu desempenho.

1. Qual comando SQL é usado para inserir novos dados em uma tabela?

A) INSERT INTO
B) ADD RECORD
C) CREATE ROW
D) NEW ENTRY

2. O que a seguinte consulta retorna?
SELECT nome FROM Alunos WHERE idade > 20;

A) Todos os nomes da tabela Alunos
B) Os nomes dos alunos com idade maior que 20
C) Os nomes e idades dos alunos com mais de 20 anos
D) A quantidade de alunos com mais de 20 anos

3. Complete a consulta para selecionar todos os alunos com idade igual a 21:

SELECT * FROM Alunos

4. Qual comando é usado para atualizar dados existentes em uma tabela?

A) MODIFY
B) ALTER
C) UPDATE
D) CHANGE

5. Complete a consulta para ordenar os alunos por nome em ordem alfabética:

SELECT * FROM Alunos

6. O que acontece se você executar um comando DELETE sem a cláusula WHERE?

A) Nada, a sintaxe está incorreta
B) Apenas a primeira linha é excluída
C) Todas as linhas da tabela são excluídas
D) A tabela é excluída do banco de dados

7. Complete a consulta para contar o número total de alunos:

FROM Alunos;

8. Qual tipo de JOIN retorna todos os registros da tabela à esquerda, mesmo sem correspondência na direita?

A) INNER JOIN
B) LEFT JOIN
C) RIGHT JOIN
D) FULL JOIN
